1. Excel යනු කුමක්ද? — මූලික හැඳින්වීම

🖥️ Excel Window එකේ ප්‍රධාන කොටස්

1

Ribbon (ரிப்பன்)

ඉහළින් ඇති toolbar එක. Home, Insert, Formulas, Data ආදී tabs ඇත. සියලු features මෙහිදී.

2

Name Box

ඉහළ වමේ කුඩා box එක — ඔබ select කළ cell එකේ address පෙන්වයි. (උදා: A1, B5, C10)

3

Formula Bar

Name Box ළඟ ඇති දිග box එක. Select කළ cell එකේ ඇති සූත්‍රය හෝ data පෙන්වයි.

4

Columns & Rows

Column = සිරස් (A, B, C...). Row = තිරස් (1, 2, 3...). Cell Address = Column + Row (A1, B3, D7)

5

Sheet Tabs

පහළින් ඇති Sheet1, Sheet2... tabs. එකිනෙකා file එකක් තුළ pages ලෙස සිතන්න.

💡 ප්‍රධාන කරුණ: Excel හි සෑම data piece එකක්ම Cell එකක ගබඩා වේ. Cell = Row + Column හමු වන ස්ථානය. A1 = A column + Row 1.

3. Data Entry — Data ඇතුල් කිරීම
Numbers, Text, Dates නිවැරදිව ඇතුළු කිරීම

🔢 Numbers (ඉලක්කම්)

Cell click කර number type කරන්න. Default ලෙස දකුණට align වේ.

Example
A1 ← 5000 A2 ← 12500.50 A3 ← -800
❌ Number සමග spaces හෝ letters දාන්නේ නැහැ — Excel text ලෙස හඳුනාවි.

📝 Text (වචන)

Employee name, product name ආදිය. Default ලෙස වමට align වේ.

Example
A1 ← Nisha A2 ← Export Items A3 ← January

📅 Dates (දිනයන්)

Basic

Excel Date format: MM/DD/YYYY හෝ DD-MM-YYYY

Date Entry
A1 ← 01/06/2025 ← June 1, 2025 A2 ← 15-06-2025 ← June 15, 2025
💡 Tip: Ctrl + ; (semicolon) 누르면today's date auto insert වේ!

⚡ AutoFill — ස්වයංක්‍රීය පිරවීම

AutoFill Examples
1, 2, 3... → ඉලක්කම් series Mon, Tue... → දිනවල names Jan, Feb... → මාස names 2025, 2026... → අවුරුදු

4. සූත්‍ර (Formulas) — ගණනය කිරීම
Excel ගණිත — = ලකුණෙන් ආරම්භ

📌 Rules — නීතිය

✅ සෑම formula එකක්ම = (Equal sign) ලකුණෙන් ආරම්භ කළ යුතුය!
Basic Operators
= A1 + B1 ← Add (එකතු) = A1 - B1 ← Subtract (අඩු) = A1 * B1 ← Multiply (ගුණ) = A1 / B1 ← Divide (බෙදීම) = A1 ^ 2 ← Power (A1 හි වර්ගය)

5. ප්‍රධාන Functions
ගිණුම්කාරිණියකට වැදගත්ම Excel functions

➕ SUM — එකතු කිරීම

Most Used
Syntax
=SUM(A1:A10) ← A1 සිට A10 දක්වා =SUM(A1,B1,C1) ← Selected cells =SUM(A:A) ← A column සියල්ල
A B (Sales - Rs.)
January 125,000
February 98,500
March 145,200
Total =SUM(B2:B4) → 368,700

📊 AVERAGE — සාමාන්‍ය

Syntax
=AVERAGE(B2:B4) ← 368,700 ÷ 3 = 122,900

🔢 COUNT / COUNTA — ගණන් කිරීම

Syntax
=COUNT(A1:A20) ← Numbers ගණන =COUNTA(A1:A20) ← සියලු filled cells =COUNTIF(A1:A20,"T-Shirt") ← "T-Shirt" කීවාරක් ද?
💡 Export Example: =COUNTIF(C2:C50,"Shipped") — "Shipped" orders ගණන ගනී.

📈 MAX / MIN — ඉහළම / පහළම

Syntax
=MAX(B2:B12) ← හොඳම sales month =MIN(B2:B12) ← අඩුම sales month

🤔 IF — Condition (නිබන්ධනය)

Important
Syntax
=IF(condition, value_if_true, value_if_false)

Clothing export ප්‍රායෝගික:

Export Company Examples
=IF(D2>=100000,"Target Achieved","Below Target") ← Sales 100,000+ නම් "Target Achieved" =IF(B2>500,"Bulk Order","Normal Order") ← Qty 500+ නම් "Bulk Order" =IF(E2="Paid","✓ Clear","⚠ Pending") ← Payment status check

🔍 VLOOKUP — Data සොයා ගැනීම

Advanced
Syntax
=VLOOKUP(lookup_value, table, col_index, 0)

Product code දී price සොයා ගැනීමට:

A (Code) B (Product) C (Price)
P001 T-Shirt 850
P002 Jeans 2,200
P003 Jacket 4,500
Usage
=VLOOKUP("P002", A2:C4, 3, 0) ← P002 හි price → 2,200 =VLOOKUP(G2, A2:C4, 2, 0) ← G2 හි code ට product name
💡 col_index: Table එකේ 1st column = 1, 2nd = 2, 3rd = 3. අවසාන 0 = Exact Match

💰 SUMIF — Condition එකකට Sum

Accounting Key
Syntax
=SUMIF(range, criteria, sum_range)
Export Examples
=SUMIF(C2:C50, "Jacket", D2:D50) ← Jacket sales එකතුව =SUMIF(B2:B50, "January", E2:E50) ← January month sales =SUMIF(F2:F50, "Pending", G2:G50) ← Pending payments total

🔤 Text Functions

Useful Text Functions
=UPPER("nisha") → NISHA =LOWER("NISHA") → nisha =PROPER("nisha de silva") → Nisha De Silva =LEN("T-Shirt") → 7 (character count) =TRIM(" Nisha ") → "Nisha" (spaces ඉවත්) =CONCATENATE(A1," ",B1) → words join =A1&" "&B1 → Same as above =LEFT("PO-12345",2) → "PO" =MID("PO-12345",4,5)→ "12345"

7. Charts — ප්‍රස්ථාර
Data graphs ලෙස දෘශ්‍යමාන කිරීම

📈 Chart හදන විදිහ — Step by Step

1

Data Select කරන්න

Chart කළ යුතු data range select කරන්න (headers සමගින්). Shift+Click හෝ drag.

2

Insert → Chart

Ribbon → Insert tab → Charts group → Chart type select කරන්න.

3

Chart Type තෝරන්න

Column, Bar, Line, Pie ආදිය. Export data සඳහා Column Chart / Line Chart suitable.

4

Chart Title Add කරන්න

Chart click → Design tab → "Monthly Sales Report 2025" ආකාරයෙන් title add.

📊 Column Chart

Monthly sales compare කරන්න. Bar chart = Same but horizontal.

Best for: Sales comparison, Product performance

🥧 Pie Chart

Total sales හි product share % පෙන්වන්න. Parts of a whole.

Best for: Market share, Expense breakdown

📈 Line Chart

Trend over time. Month by month sales trend.

Best for: Growth trends, Over-time analysis

🔲 Pivot Table

Data summarize කිරීමට. Insert → PivotTable → fields drag.

Best for: Big data summary reports

9. Sort & Filter — ක්‍රමගත කිරීම
Data organize කිරීම හා නිශ්චිත data සොයා ගැනීම

⬆️ Sort (ക्रमबद्ध) කිරීම

1

Data range select

Sort කිරීමට අවශ්‍ය column හෝ entire table select කරන්න.

2

Data → Sort

Column choose කර A→Z (ascending) හෝ Z→A (descending) select.

💡 Example: Sales column Sort Z→A = Highest sales first. Date Sort A→Z = Oldest to Newest.

🔎 Filter (Autofilter) — ෆිල්ටර්

1

Header row click

Data ඇති table header row click කරන්න.

2

Data → Filter (Ctrl+Shift+L)

Headers වල dropdown arrows appear වේ. Click → filter options.

3

Filter apply කරන්න

Specific items check/uncheck. Number filters: Greater Than, Between ආදිය.

💡 Export Example: Product column filter → "T-Shirt" only → T-Shirt orders மட்டும் show. Status filter → "Pending" → pending orders.

11. Pro Tips & Best Practices
ගිණුම්කාරිණියකගේ Excel Workflow

✅ File Save Best Practices

  • සෑම විටම Ctrl+S → File save කරන්න
  • File name: SalesReport_June2025.xlsx ආකාරයෙන් date include කරන්න
  • Excel file format: .xlsx (default, best)
  • Cloud backup: OneDrive හෝ Google Drive sync recommended
  • Weekly backup copy separate folder එකේ save කරන්න

📐 Data Entry Rules — Error ගෙදර නොයවන්න

  • Numbers සමග currency symbols (Rs., $) type නොකරන්න — formatting use කරන්න
  • Dates consistent format එකෙන් ඇතුළු කරන්න (MM/DD/YYYY හෝ DD-MM-YYYY)
  • Empty rows/columns table middle හි නොතබන්න
  • Column headers Row 1 හිම තබන්න (A1 සිට)
  • Spaces extra ලෙස නොදමන්න — =TRIM() use කරන්න
