➕ SUM
Most Used
=SUM(A1:A10) ← A1 සිට A10 =SUM(A1,B1,C1) ← Selected cells =SUM(A:A) ← A column සියල්ල
AB (Sales - Rs.)
January125,000
February98,500

Total=SUM(B2:B4) → 368,700
📊 AVERAGE
=AVERAGE(B2:B4) → 122,900
📈 MAX / MIN
=MAX(B2:B12) =MIN(B2:B12)
🔢 COUNT / COUNTIF
=COUNT(A1:A20) ← Numbers ගණන =COUNTA(A1:A20) ← Filled cells =COUNTIF(A1:A20,"T-Shirt") ← "T-Shirt" ගණන
💡 =COUNTIF(C2:C50,"Shipped") → Shipped orders ගණන
🤔 IF — Condition
Important
Syntax
=IF(condition, if_true, if_false)
Export Examples
=IF(D2>=100000,"Target ✓","Below") ← Sales 100k+ නම් =IF(B2>500,"Bulk","Normal") ← Qty 500+ නම් Bulk =IF(E2="Paid","✓ Clear","⚠ Pending") ← Payment status
🔍 VLOOKUP
Advanced
Syntax
=VLOOKUP(value, table, col, 0)
A (Code)B (Product)C (Price)
P001T-Shirt850
P002Jeans2,200
P003Jacket4,500
=VLOOKUP("P002", A2:C4, 3, 0) ← P002 price → 2,200 =VLOOKUP(G2, A2:C4, 2, 0) ← G2 code → product name
💡 col: 1st column=1, 2nd=2, 3rd=3 · අවසාන 0 = Exact Match
💰 SUMIF
Accounting Key
Syntax
=SUMIF(range, criteria, sum_range)
Export Examples
=SUMIF(C2:C50,"Jacket",D2:D50) ← Jacket sales total =SUMIF(B2:B50,"January",E2:E50) ← January sales =SUMIF(F2:F50,"Pending",G2:G50) ← Pending payments
🔤 Text Functions
=UPPER("nisha") → NISHA =LOWER("NISHA") → nisha =PROPER("nisha silva") → Nisha Silva =LEN("T-Shirt") → 7 =TRIM(" Nisha ") → "Nisha" =A1&" "&B1 → Join cells =LEFT("PO-12345",2) → "PO" =MID("PO-12345",4,5) → "12345"

Pro Tips Best Practices & Common Errors
✅ File Save Best Practices
  • සෑම විටම Ctrl+S save කරන්න
  • File name: SalesReport_June2025.xlsx — date include
  • Format: .xlsx (best)
  • Cloud: OneDrive / Google Drive sync
  • Weekly backup — separate folder
📐 Data Entry Rules
  • Numbers සමග Rs., $ type නොකරන්න — formatting use
  • Dates consistent format — MM/DD/YYYY
  • Table middle හි empty rows නොදමන්න
  • Headers Row 1 — A1 සිට
  • Extra spaces — =TRIM() use
